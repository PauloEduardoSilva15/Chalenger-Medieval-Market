# Chalenger-Medieval-Market

## Descrição

Esse projeto faz parte de uma das etapas do processo seletivo da Apple Developer Accademy de 2025. O exercício proposto era criar um programa de terminal escrito em swift, na qual devemos utilizar os conhecimentos obtidos no workshop para desenvolver esse programa no tema de RPG.

O projeto se trata de um simulador de lojinha de RPG, na qual vc jogador pode comprar items na loja, esses items que foram comprados ficarão armazenados no inventário do jogador. Você poderá vender esses itemss e eles irão sumir do inventário. Também haverá uma sessão de apostas que se baseará em o jogador adivinhar o numero sorteado no dado, caso vença ele ganha 25 de moedas do jogo, caso perca irá subtrair 50 de moedas.

## Como executar o projeto
Para rodar o projeto necessitará do swift e do programa Xcode para rodar o programa.

Você clona o repositório e abra o arquivo *MedievalMarket.xcodeproj* na pasta *Chalemger-Medieval-Market*, abrirá o projeto no Xcode e você podera executar o projeto.
